# SiteMirror

**Author: Cyber Anxhu**

> Website Cloning & Phishing Assessment Tool for Authorized Security Testing 


## Features

- **Clone any website** — HTML, CSS, JavaScript, images, fonts, all assets
- **Inject credential capture** — "Session Expired" overlay that prompts for login
- **Real-time capture** — See credentials appear instantly in your terminal
- **IP geolocation** — Automatically get city, country, ISP of the target
- **Public URL** — Auto-tunnel via Cloudflare or ngrok
- **Cross-platform** — Works on Linux, macOS, and Termux (Android)

git clone https://github.com/YOUR_USERNAME/SiteMirror.git
